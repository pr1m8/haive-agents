Haive Agents Examples
=====================

This gallery showcases various agent implementations and patterns in the Haive framework.

Examples include:

- **Simple Agents**: Basic agent usage patterns
- **Multi-Agent Systems**: Coordinated agent workflows  
- **Structured Output**: Agents with typed outputs
- **Tool Integration**: Agents using various tools
- **Advanced Patterns**: Complex agent architectures

Each example demonstrates real-world usage with working code that you can run and modify.
