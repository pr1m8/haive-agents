# Haive Agent Galleries - Learn by Example

**Welcome to the Haive Agent Galleries!** 🎆

This is your hands-on learning center for mastering Haive agents. Each gallery is designed to take you from beginner to expert through practical, runnable examples.

## 🎯 Learning Path

Follow this progression for the best learning experience:

### 🌱 **Beginner Gallery** (`beginner/`)

**Start here if you're new to Haive**

- Simple agent creation and conversations
- ReactAgent with basic tools
- Understanding agent reasoning
- **Time**: 30 minutes

### 🌿 **Intermediate Gallery** (`intermediate/`)

**Ready for more complex patterns**

- Multi-agent coordination
- Structured output generation
- Plan and execute workflows
- **Time**: 1-2 hours

### 🌲 **Advanced Gallery** (`advanced/`)

**Master complex agent systems**

- Dynamic agent networks
- Custom workflow patterns
- Performance optimization
- **Time**: 2-4 hours

### 🎮 **Games Gallery** (`games/`)

**Fun with AI game-playing agents**

- Tic-tac-toe AI
- Chess strategy agents
- Multi-player game coordination
- **Time**: 1-2 hours

## 🚀 Quick Start

```bash
# Start with the beginner gallery
cd packages/haive-agents/galleries/beginner
poetry run python simple_agent_tutorial.py

# When ready, move to intermediate
cd ../intermediate
poetry run python multi_agent_coordination.py

# Advanced patterns
cd ../advanced
poetry run python dynamic_agent_networks.py

# Have fun with games!
cd ../games
poetry run python tic_tac_toe_ai.py
```

## 🛠️ Prerequisites

**Required**:

- Python 3.12+
- Haive framework installed (`pip install haive`)
- OpenAI or Anthropic API key configured

**Optional but recommended**:

- Basic understanding of async/await in Python
- Familiarity with type hints
- Experience with LLM APIs
